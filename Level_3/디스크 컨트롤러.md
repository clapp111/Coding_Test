## 디스크 컨트롤러 (Lv.3)

https://school.programmers.co.kr/learn/courses/30/lessons/42627

### 기록

1. **작업 스케줄링 + 우선 순위 큐**
    
    이 문제는 우선 순위 디스크 컨트롤러라는 가상의 장치를 제시하면서 동작 방식을 알려주고 이를 구현하라는 문제이다.
    
    요청된 모든 작업들 중에서 소요 시간이 가장 짧은 작업을 먼저 처리하는 방식임을 알 수 있다.
    
    따라서 두 가지 기준으로 작업의 정보를 정리해야 했다.
    
    1. 아직 처리하지 않은 작업들 중에서 요청 시각이 빠른 순서대로 확인
    2. 현재 시점에서 실행 가능한 작업들 중에서 소요 시간이 짧은 순서대로 선택
    
    그래서 먼저 jobs를 요청 시각 기준으로 정렬하고, 현재 시간 이하로 들어온 작업들을 우선순위 큐에 넣고, 그 중 가장 짧은 작업을 꺼내 처리하는 방식으로 코드를 설계했다.
    
    두 개의 비교 함수를 만들고, 작업을 배치하는 방식을 잘 이해한다면 해결할 수 있는 문제였다.
    
2. **막혔던 점**
    
    비교 함수를 작성하는데 어려움이 있었다. 이전에는 단순히 하나의 원소를 가진 배열을 비교했지만, 이번에는 2~3개의 원소를 가진 자료구조 내에서 비교를 하는 함수를 짜야했기 때문에 사용한 적이 없었던 C++ 문법을 사용했다. 이 부분은 구글링을 통해 공부하고 적용했다.
    

### 제출 코드

```cpp
#include <string>
#include <vector>
#include <queue>
#include <algorithm>

using namespace std;

struct cmp{
    bool operator()(const vector<int>& a, const vector<int>& b){
        if(a[0] == b[0]){
            if(a[1] == b[1]){
                return a[2] > b[2];
            }
            return a[1] > b[1];
        }
        return a[0] > b[0];
    }
};

int solution(vector<vector<int>> jobs) {
    sort(jobs.begin(),jobs.end(),[](const vector<int>& a , const vector<int>& b){
        if(a[0] == b[0]){
            return a[1] < b[1];
        }
        return a[0] < b[0];
    });

    priority_queue<vector<int>, vector<vector<int>>,cmp> pq;
    int n = jobs.size();
    int i = 0;
    int complete = 0;
    int current_time = 0;
    int sum_of_return_time = 0;
    while(complete < n){
        while(i < n && jobs[i][0] <= current_time){
            pq.push({jobs[i][1], jobs[i][0], i++});
        }
        if(pq.empty()){
            current_time = jobs[i][0];
            continue;
        }
        auto top = pq.top();
        pq.pop();
        current_time += top[0];
        sum_of_return_time += current_time - top[1];
        complete++;
    }

    return sum_of_return_time / n;
}
```
