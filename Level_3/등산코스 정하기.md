
## 등산코스 정하기 (Lv.3)

https://school.programmers.co.kr/learn/courses/30/lessons/118669

### 기록

1. **다익스트라 알고리즘**
    
    해당 문제는 최단 거리를 찾는 다익스트라 알고리즘의 변형 문제로 볼 수 있다. 거리의 합을 최소화하는 최단거리 문제가 아니라, 경로의 비용을 max(간선 가중치)로 정의한 문제이다. 따라서 다익스트라 알고리즘의 기본 개념과 활용 능력을 묻는 문제라고 생각했다.
    
    DFS를 사용해서 모든 경로를 찾을 수도 있지만, 노드와 간선의 수가 많아 진다면 가능한 경로의 수가 기하급수적으로 많아지기에 시간 초과가 발생할 수 있다.
    
2. **우선 순위 큐와 인접 리스트의 사용**
    
    우선 순위 큐와 인접 리스트를 사용하여 다음으로 올 간선에 대한 정보를 조회할 수 있게 했다.
    
    기본적으로 intensity를 저장하기 위해 dijk 배열을 선언하였다.
    
    또한 pair<int,int>를 사용하여 우선 순위 큐에는 `{현재까지의 intensity, 현재 노드}`를 저장하고, 인접 리스트에는 `{다음 노드, 간선 가중치}`를 저장한다.
    
3. **고쳐야할 점**
    - 다익스트라 알고리즘은 강의 시간에도 이해하기 어려웠던 내용인 만큼, 시간이 지난 지금 다시 다루기 어려웠다. 처음에는 DFS로 설계를 하려고 했으나. 이미 LV.2에서 DFS문제를 많이 다뤘으니 이번에는 다익스트라로 해결하고 싶었다.
    - 인접리스트, 우선 순위 큐, 원소 업데이트 등등 각 개념의 상호작용을 이해하는 데 어려움이 있었고 해결하는 데 오랜 시간이 결렸다.
    - 블로그를 보고 해결 방법을 얻었고, 세부적인 설계는 GPT의 도움을 받았다. 특히 `if (intensity > dijk[current_node]) continue;` 부분은 생각하지도 못한 로직이라 더 공부가 필요하다는 것을 느꼈다.

### 제출 코드

```cpp
#include <string>
#include <vector>
#include <algorithm>
#include <queue>
#include <climits>

using namespace std;

vector<int> solution(int n, vector<vector<int>> paths, vector<int> gates, vector<int> summits) {
    // 그래프를 인접 리스트로 표현
    vector<vector<pair<int, int>>> adj(n + 1);
    for (const auto& path : paths) {
        adj[path[0]].push_back({path[1], path[2]});
        adj[path[1]].push_back({path[0], path[2]});
    }

    // 우선순위 큐(작은 값 우선)를 사용하여 낮은 intensity를 먼저 처리
    priority_queue<pair<int, int>, vector<pair<int, int>>, greater<pair<int, int>>> pq;
    vector<int> dijk(n + 1, INT_MAX);
    vector<bool> is_summit(n + 1, false);
    vector<bool> is_gate(n + 1, false);
    for(int summit : summits) is_summit[summit] = true;
    for(int gate : gates){
        pq.push({0, gate}); // {intensity, current_node}
        dijk[gate] = 0; // 출입구의 intensity는 0으로 시작
        is_gate[gate] = true;
    }
    while(!pq.empty()){
        auto [intensity, current_node] = pq.top();
        pq.pop();
        if (intensity > dijk[current_node]) continue; // 이미 더 낮은 intensity로 방문한 노드라면 무시
        if(is_summit[current_node]) continue; // 산봉우리라면 더 이상 탐색하지 않음
        for(const auto& [next_node, weight] : adj[current_node]){
            if(is_gate[next_node]) continue; // 출입구는 경유지로 사용하지 않음
            int new_intensity = max(intensity, weight); // 현재까지의 intensity와 다음 간선의 가중치 중 큰 값을 선택
            if(new_intensity < dijk[next_node]){ // 더 낮은 intensity로 방문할 수 있다면 업데이트
                dijk[next_node] = new_intensity;
                pq.push({new_intensity, next_node});
            }
        }
    }
    sort(summits.begin(), summits.end()); // 동률일 때 번호가 작은 봉우리를 선택함
    int bestSummit = summits[0];
    int bestIntensity = dijk[bestSummit];
    for (int s : summits) {
        if (dijk[s] < bestIntensity) {
            bestIntensity = dijk[s];
            bestSummit = s;
        }
    }
    return {bestSummit, bestIntensity};
}
```
