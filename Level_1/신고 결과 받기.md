### 기록

1. **sort() + unique() + erase() 를 사용한 vector에서의 중복 제거**
    - 정렬을 한다: sort()
        - sort(start_index, end_index);
    - 연속된 중복 원소를 vector의 제일 뒷부분(쓰레기 값)으로 보낸다: unique()
        - unique()는 반환값은 vector 쓰레기 값의 첫번째 위치가 된다.
        - unique(start_index, end_index);
    - 쓰레기 값을 모두 지운다: erase()
        - vector.erase(start_index, end_index);

### 제출 코드

```cpp
#include <string>
#include <vector>
#include <sstream>
#include <algorithm>
#include <unordered_map>

using namespace std;

vector<int> solution(vector<string> id_list, vector<string> report, int k) {
    int n = (int)id_list.size();
    vector<int> reported(n, 0);
    vector<int> answer(n, 0);
    unordered_map<string, int> um;
    
    sort(report.begin(), report.end());
    report.erase(unique(report.begin(), report.end()), report.end());
    
    for(int i = 0; i < n; i++) um[id_list[i]] = i;
    for(string rep : report){
        stringstream ss(rep);
        string a, b;
        ss >> a >> b;
        reported[um[b]]++;
    }
    for(string rep : report){
        stringstream ss(rep);
        string a, b;
        ss >> a >> b;
        if(reported[um[b]] >= k) answer[um[a]]++;
    }
    return answer;
}
```
