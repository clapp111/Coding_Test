### 기록

1. **다익스트라 알고리즘 X 3**
    
    이 문제는 가중치가 있는 그래프에서 최소 비용을 구하는 문제이므로 다익스트라를 사용해야한다는 생각이 들었다.
    
    특히 하게도 분기점까지는 같이 가되, 이후에는 흩어진다는 특징이 있다.
    
    간단하게 생각해보면 분기점에서 A의 집, B의 집, 시작점 S까지 세 갈래로 나뉘어 지며, 이 세 갈래의 비용의 합이 최소가 되는 분기점을 고르면 된다.
    
    따라서 A의 집, B의 집, 시작점 S에서 각각 다익스트라를 돌리고 임의의 분기점 K 까지의 거리를 구하고 비교하면 되는 문제이다.
    
2. **우선 순위 큐**
    
    가중치가 있는 그래프이며 사용하는 알고리즘이 다익스트라이므로, 우선 순위 큐가 일반적으로 사용된다.
    
    그래프를 인접 리스트로 구성하고, 각 정점에서 연결된 정점과 비용을 함께 저장했다.
    
3. **막혔던 점**
    - 합승과 분리라는 특징 때문에 처음에는 혼란스러웠다. 나는 순서대로 문제를 해결하는 경향이 있기에, 합승의 경우를 처리하고 이후의 경우를 처리하는 복잡한 과정을 먼저 떠올렸다. 가장 직관적이지만 문제 해결을 위해서는 고민해봐야하는 방식이다. 문제의 성질을 먼저 완벽히 이해하는 것이 우선이다.

### 제출 코드

```cpp
#include <string>
#include <vector>
#include <queue>
#include <climits>

using namespace std;

vector<int> dijk(int start, const vector<vector<pair<int, int>>> &graph) {
    vector<int> dist(graph.size(), INT_MAX);
    dist[start] = 0;
    priority_queue<pair<int, int>, vector<pair<int, int>>, greater<pair<int, int>>> pq;
    pq.push({0, start});

    while (!pq.empty()) {
        auto [cost, node] = pq.top();
        pq.pop();

        if (cost > dist[node]) continue;

        for (const auto &[next_node, next_cost] : graph[node]) {
            int new_cost = cost + next_cost;
            if (new_cost < dist[next_node]) {
                dist[next_node] = new_cost;
                pq.push({new_cost, next_node});
            }
        }
    }

    return dist;
}

int solution(int n, int s, int a, int b, vector<vector<int>> fares) {
    vector<vector<pair<int, int>>> graph(n + 1);

    for (const auto &fare : fares) {
        int u = fare[0];
        int v = fare[1];
        int cost = fare[2];
        graph[u].push_back({v, cost});
        graph[v].push_back({u, cost});
    }

    vector<int> dist_from_s = dijk(s, graph);
    vector<int> dist_from_a = dijk(a, graph);
    vector<int> dist_from_b = dijk(b, graph);

    int min_cost = INT_MAX;
    for(int i = 1; i <= n; i++) {
        min_cost = min(min_cost, dist_from_s[i] + dist_from_a[i] + dist_from_b[i]);
    }
    
    return min_cost;
}
```
