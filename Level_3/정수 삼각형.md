
## 정수 삼각형 (Lv.3)

https://school.programmers.co.kr/learn/courses/30/lessons/43105

### 기록

1. **동적계획법**
    
    구조를 보면 트리 구조에서 가장 큰 누적 합을 구하는 문제라고 볼 수 있다. 
    
2. **Top-down(하향식) vs Bottom-up(상향식)**
    
    이런 유형의 문제를 해결하기 위해서는 하향식과 상향식, 둘 중 하나를 골라서 해결할 수 있다.
    
    하향식은 직관적이고 문제에서 의도하는 흐름을 타면서 해결할 수 있지만, 비교하는 조건을 잘 설계해야한다.
    
    상향식은 다소 직관적이지는 않을지라도, 각 단계에서 결과를 저장하는 방법이 간단하다. 단순히 자식 노드의 값을 비교해서 더 큰 값을 부모 노드에 더하면 된다.
    
3. **고쳐야할 점**
    - 처음에는 하향식으로 설계를 하다가 비교하는 방법이 복잡해질 것 같아서 상향식으로 바꾸었다. 이와 비슷한 누적합 문제를 풀 때는 상향식을 우선적으로 생각해보자.

### 제출 코드

```cpp
#include <string>
#include <vector>

using namespace std;

int solution(vector<vector<int>> triangle) {
    int n = triangle.size();
    for (int i = n - 1; i > 0; i--){
        for(int j = 0; j < triangle[i].size() - 1; j++){
            triangle[i - 1][j] += max(triangle[i][j], triangle[i][j + 1]);
        }
    }
    return triangle[0][0];
}
```
