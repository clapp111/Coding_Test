
## 네트워크 (Lv.3)

https://school.programmers.co.kr/learn/courses/30/lessons/43162

### 기록

1. **BFS 알고리즘**
    
    FIFO(선입선출) 방식의 큐를 배운 만큼, 이를 사용해서 BFS방식으로 해결하고 싶었다.
    
    해당 노드를 방문 했는지 여부를 확인하는 visited 배열을 만들어서 방문한 노드인지 체크한다. 
    
    큐에서 노드를 pop할 때 만약 방문하지 않은 노드라면 이어진 간선을 큐에 삽입하고, 해당 노드를 방문했다고 기록한다.
    
    이렇게 큐가 비워질때까지 반복하면 한 네트워크가 완성된다.
    

### **더 생각해볼 점들**

- DFS도 동일하게 방문한 노드인지 확인할 배열이 필요하다. 설계해 본다면, 해당 노드에서 갈 수 있는 다음 노드를 재귀 함수의 인자로 준다. 또한 해당 함수 처음에서 인자로 받은 노드가 이미 방문한 노드인지 확인하여 함수를 종료시키면 될 듯 싶다.

### 제출 코드

```cpp
#include <string>
#include <vector>
#include <queue>

using namespace std;

int solution(int n, vector<vector<int>> computers) {
    int count = 0;
    vector<bool> visited(n, false);

    for (int i = 0; i < n; i++) {
        if (visited[i]) continue;
        
        count++;
        queue<int> q;
        q.push(i);
        visited[i] = true;

        while (!q.empty()) {
            int current = q.front();
            q.pop();
            for (int j = 0; j < n; j++) {
                if (computers[current][j] == 1 && !visited[j]) {
                    visited[j] = true;
                    q.push(j);
                }
            }
        }
    }
    return count;
}
```
