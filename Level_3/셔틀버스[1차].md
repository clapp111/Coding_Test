### 기록

1. **정렬 기반 시뮬레이션**
    
    이 문제는 각 크루의 도착 시간을 오름차순으로 정렬하고, 각 버스가 도착할 때마다 탈 수 있는 인원을 순서대로 태우는 것이다.
    
    알고리즘을 사용하는 문제보다는, 단순하게 주어진 시간표를 정렬하고 버스 운행 과정을 그대로 시뮬레이션하는 구현 문제에 가깝다.
    
    크루의 도착 시간과 버스의 도착 시간을 분 단위로 변환하고 우선순위 큐를 사용해서 문제를 해결했다.
    
2. **막혔던 점**
    
    정답은 “콘이 가장 늦게 도착할 수 있는 시간”이기 떄문에 모든 버스를 다 확인한 뒤 마지막 버스가 꽉 찼는지 여부를 기준으로 판단해야 했다.
    
    마지막 버스에 자리가 남아 있다면, 콘은 버스 도착 시각에 와도 된다.
    
    마지막 버스가 가득 찼다면, 마지막으로 탄 크루보다 1분 일찍 와야한다.
    
    “가장 늦게 도착할 수 있는 시간”을 구하기 위해 설정해야 할 변수를 정하는게 중요했다.
    

### 제출 코드

```cpp
#include <string>
#include <vector>
#include <queue>

using namespace std;

string minute_to_time(int time){
    string answer = "";

    if(time/60 < 10) answer += "0" + to_string(time / 60);
    else answer += to_string(time / 60);
    
    answer += ":";
    
    if(time%60 < 10) answer += "0" + to_string(time % 60);
    else answer += to_string(time % 60);

    return answer;
}

string solution(int n, int t, int m, vector<string> timetable) {
    priority_queue<int, vector<int>, greater<int>> pq;

    for(string time : timetable){
        pq.push(stoi(time.substr(0,2)) * 60 + stoi(time.substr(3,2)));
    }

    int bus_arrival = 540;
    int last_crew = 0;
    int count = 0;

    for(int i = 0; i < n ; i++){
        bus_arrival = 540 + i * t;
        count = 0;
        while(!pq.empty() && count < m && pq.top() <= bus_arrival){
            count++;
            last_crew = pq.top();
            pq.pop();
        }
    }

    if (count < m){
        return minute_to_time(bus_arrival);
    }
    else {
        return minute_to_time(last_crew - 1);
    }
}
```
