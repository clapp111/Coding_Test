## 가장 먼 노드 (Lv.3)

### 기록

1. **BFS + queue**
    
    이 문제는 가중치가 없는 양방향 그래프에서 1번 노드로부터 가장 멀리 떨어진 노드의 개수를 구하는 문제이다.
    
    모든 간선의 비용은 동일하므로 다익스트라보다 BFS가 더 적절하다.
    
    시작 노드인 1번에서부터 BFS를 수행하면, 먼저 방문한 순서대로 거리가 확정된다.
    
    따라서 어떤 노드를 처음 방문하는 순간의 거리가 곧 1번 노드로부터의 최단 거리가 된다.
    
    만약 DFS를 사용했다면, 특정 노드에 도달했을 때 값을 비교해야하는 로직이 추가되므로 BFS가 더 적절하다고 할 수 있다.
    
2. **고쳐야 할 점**
    - 굳이 visited 배열을 사용하지 않아도 dist 배열의 값의 정보로 방문 여부를 알 수 있다.

### 제출 코드

```cpp
#include <string>
#include <vector>
#include <queue>

using namespace std;

int solution(int n, vector<vector<int>> edge) {
    vector<int> dist(n + 1, 1e9);
    vector<bool> visited(n + 1, false);
    vector<vector<int>> graph(n + 1);
    queue<int> q;

    for(const auto& e : edge){
        int node_1 = e[0];
        int node_2 = e[1];
        graph[node_1].push_back(node_2);
        graph[node_2].push_back(node_1);
    }
    q.push(1);
    dist[1] = 0;
    visited[1] = true;
    while(!q.empty()){
        int current = q.front();
        q.pop();
        for(const auto& next : graph[current]){
            if(!visited[next]){
                dist[next] = dist[current] + 1;
                visited[next] = true;
                q.push(next);
            }
        }
    }

    int max_dist = 0;
    int max_count = 0;
    for(int i = 2; i <= n; i++){
        if(dist[i] == 1e9) continue;
        if(dist[i] > max_dist){
            max_dist = dist[i];
            max_count = 1;
        } else if(dist[i] == max_dist){
            max_count++;
        }
    }
    return max_count;
}
```
