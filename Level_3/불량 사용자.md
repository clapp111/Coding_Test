
## 불량 사용자 (Lv.3)

### 기록

1. **매칭되는 유저를 각 불량 유저 이름 별로 정리**
    
    이 문제를 봤을 때 가능한 모든 경우를 구해야 하고 조건이 엄격하기 때문에 DFS + 조건 구현 로직을 통해 해결해야겠다고 생각했다. 먼저 2차원 vector를 만들어서 각 불량 유저 이름이 매칭될 수 있는 일반 유저의 인덱스를 담는다.
    
2. **DFS + set 으로 중복 결과 제거** 
    
    위에서 구한 2차원 vector와 방문 여부를 나타내는 visited 배열을 가지고 DFS를 구현했다. 만약 해당 루트가 완료되었다면, set에 해당 루트의 visited를 넣는다. 뒤에서 순서만 바뀌고 내용은 일치한 루트가 나오더라고 visited는 bool타입으로 선언되었기 때문에 set안에서는 자동으로 중복 제거가 된다.
    
3. **고쳐야 할 점**
    - 처음엔 multimap을 사용해서 (1)을 완성시키고 싶었지만, 각 불량 유저 이름 별로 조회를 해야하기 때문에 multimap은 적절하지 않았다. 어차피 (1)에서는 모든 인덱스를 돌아야 하기 때문에 간단하게 2차원 vector를 사용하는 것이 적절한 판단이었다.
    - 결과에서 중복을 제거할 수 있는 방법을 한참 생각했다. 분명히 set을 사용하는 것이 맞지만 어디서 어떻게 사용해야할지 뇌가 멈춰버렸다. 더 다양한 자료구조에 대한 이해가 필요해 보인다.
    

### 제출 코드

```cpp
#include <iostream>
#include <string>
#include <vector>
#include <set>

using namespace std;

bool check(const string& user, const string& banned) {
    if (user.size() != banned.size()) return false;

    for (int i = 0; i < user.size(); i++) {
        if (banned[i] == '*') continue;
        if (user[i] != banned[i]) return false;
    }
    return true;
}

void dfs(int depth, const vector<vector<int>>& matches, vector<bool>& visited, set<vector<bool>>& result) {
    int n = matches.size();
    if (depth == n) {
        result.insert(visited);
        return;
    }
    int m = matches[depth].size();
    for (int i = 0; i < m; i++) {
        int user_index = matches[depth][i];
        if (!visited[user_index]) {
            visited[user_index] = true;
            dfs(depth + 1, matches, visited, result);
            visited[user_index] = false;
        }
    }
    return;
}

int main() {
    vector<string> user_id = {"frodo", "fradi", "crodo", "abc123", "frodoc"};
    vector<string> banned_id = {"fr*d*", "*rodo", "******", "******"};

    vector<bool> visited(user_id.size(), false);
    vector<vector<int>> matches(banned_id.size());
    set<vector<bool>> result;

    for (int i = 0; i < banned_id.size(); i++) {
        for (int j = 0; j < user_id.size(); j++) {
            if (check(user_id[j], banned_id[i])) {
                matches[i].push_back(j);
            }
        }
    }

    dfs(0, matches, visited, result);
    cout << result.size() << endl;

    return 0;
}
```
